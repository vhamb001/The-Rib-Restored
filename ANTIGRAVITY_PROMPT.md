# Prompt for Antigravity / AI Coding Agent

Build a simple, elegant personal website for **The Rib Restored by Shalaymah** using Next.js App Router, TypeScript, and Tailwind CSS.

## Goal

Create a personal brand website where Shalaymah can showcase her books, writings, projects, ventures, and contact information.

## Visual Direction

Use a bright royal blue, gold, ivory, and white theme. The site should look elegant, feminine, royal, peaceful, and author-centered. It should not be dark navy. Use true royal blue as the primary color.

Colors:
- Royal Blue: #0B4EDB
- Deep Royal Blue: #063A9B
- Gold: #C99A2E
- Soft Gold: #E8C766
- Ivory: #FFF9EF
- Warm White: #FFFCF7
- Soft Blush: #F4D6CE
- Text Dark: #1C1C1C

Fonts:
- Headings: Playfair Display
- Body: Inter
- Accent script: Allura

Use Google Fonts or Next Font.

## Implementation Notes

- Make the site responsive.
- Use reusable components: Header, Footer, Button, SectionTitle, BookCard, WritingCard, ProjectCard.
- Use placeholder images in `/public/images`.
- Use clean file organization.
- Keep the site simple and launch-ready.
- Avoid overcomplicating with CMS for now.
- Use static data files for books, writings, and projects.
