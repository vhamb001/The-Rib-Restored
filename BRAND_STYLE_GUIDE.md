# Brand Style Guide

## Brand Name

**The Rib Restored**

## Personal Name

**By Shalaymah**

## Brand Feel

The website should feel:
- elegant
- feminine
- royal
- peaceful
- purposeful
- author-centered
- light-filled
- premium, but not dark
- spiritual and reflective without looking too heavy

## Color Palette

### Primary Royal Blue
Use this as the main brand color.

```css
--royal-blue: #0B4EDB;
```

### Deep Royal Blue
For footer, headers, and strong contrast.

```css
--deep-royal-blue: #063A9B;
```

### Gold
For buttons, icons, dividers, logo accents.

```css
--gold: #C99A2E;
```

### Soft Gold
For subtle backgrounds and hover states.

```css
--soft-gold: #E8C766;
```

### Cream / Ivory
For background.

```css
--ivory: #FFF9EF;
```

### Warm White

```css
--warm-white: #FFFCF7;
```

### Soft Blush Accent

```css
--soft-blush: #F4D6CE;
```

### Text Black

```css
--text-dark: #1C1C1C;
```

## Font Pairing

### Headings
Use one of these:
- Cormorant Garamond
- Playfair Display
- Libre Baskerville

Recommended:
```txt
Playfair Display
```

### Body Text
Use:
```txt
Inter
```

### Script Accent Font
Use for words like “Restored,” “Let’s Connect,” or small poetic phrases:
- Great Vibes
- Allura
- Parisienne

Recommended:
```txt
Allura
```

## Logo Direction

Text-based logo:

THE  
RIB  
RESTORED

Gold typography on white or royal blue.

Optional small mark:
- crown
- open book
- feather
- restored branch
- initials “TRR”

## Image Direction

Use bright, clean imagery:
- white desk
- notebook/journal
- florals
- gold pen
- books
- soft sunlight
- royal blue mug/book
- warm author portrait
- subtle gold paint strokes

Avoid:
- overly dark navy
- cluttered prophetic graphics
- heavy church-like visuals
- too many symbols at once
